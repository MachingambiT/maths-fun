# Maths Fun! - Grade 1 Maths App

A fun and interactive addition and subtraction application designed for Grade 1 learners. All answers are less than 20, making it perfectly suited for early primary school mathematics.

## Features

- Addition, Subtraction, and Mixed mode
- Dot visualizer to help kids count along
- 4 colourful answer buttons per question
- Live score tracking with stars
- Confetti celebration for correct answers
- 10-question rounds with a results screen
- Works on phones, tablets, and computers

## How to Use

1. Open `index.html` in any web browser
2. Choose a mode: **Addition**, **Subtraction**, or **Mix**
3. Select the correct answer from the 4 buttons
4. Complete 10 questions to see your final score

## Deployment (GitHub Pages)

1. Create a new repository on [github.com](https://github.com)
2. Upload `index.html` and `README.md` to the repository
3. Go to **Settings → Pages**
4. Set source to **Deploy from a branch → main → / (root)**
5. Your live link will be available at `https://yourusername.github.io/your-repo-name/`
